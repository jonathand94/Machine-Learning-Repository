Crear el modelo de la Red Neuronal¶

Se definen los parmetros de la arquitectura, tales como el nmero de neuronas y la probabilidad de desactivacin para el regularizador.

In [12]:
num_neurons_input = 500 # Nmero de neuronas en la capa de entrada
num_neurons_hidden_1 = 250 # Nmero de neuronas en la 1 capa oculta
num_neurons_hidden_2 = 125 # Nmero de neuronas en la 2 capa oculta
prob_drop_out = 0.25 # Probabilidad de desactivar neuronas

Se crear una red neuronal con tres capas: la de entradas, la oculta y la de salida. Se usar la funcin ReLU como funcin de activacin para todas las capas excepto la de salida. Para la capa de salida, se usar una funcin llamada softmax, la cul es una extensin o generalizacin de la funcin sigmoide; usada para estimar mltiples clases.

In [13]:
# Se crea el modelo como un modelo secuencial
model = Sequential() 

# Se aade la capa de entradas
model.add(Dense(units = num_neurons_input, 
                input_dim = img_width*img_height, 
                activation = 'relu', 
                use_bias=True, 
                kernel_initializer='glorot_uniform')) 

# Se aade un regularizador para evitar sobreajuste de curvas
model.add(Dropout(prob_drop_out))

# Se aade la 1 capa oculta 
model.add(Dense(units = num_neurons_hidden_1, 
                activation = 'relu', 
                use_bias=True, 
                kernel_initializer='glorot_uniform'))

# Se aade un regularizador para evitar sobreajuste de curvas
model.add(Dropout(prob_drop_out))

# Se aade la 2 capa oculta 
model.add(Dense(units = num_neurons_hidden_2, 
                activation = 'relu', 
                use_bias=True, 
                kernel_initializer='glorot_uniform'))

# Se aade la capa de salidas
model.add(Dense(units = num_classes, activation = 'softmax'))
